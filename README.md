# 🚀 VoyageAI

Autonomous Multi-Agent Travel Planning System using LangGraph and Generative AI.

## Overview

VoyageAI is an AI-powered travel assistant where multiple specialized AI agents collaborate to create personalized travel plans.

## Features

- Multi-Agent AI workflow
- Destination recommendation
- Budget optimization
- Weather analysis
- Activity planning
- AI-generated itinerary
